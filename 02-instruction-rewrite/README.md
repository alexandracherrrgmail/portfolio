# Улучшение некачественной инструкции

## Задача
Взять плохо структурированный, запутанный текст и переработать в понятную пошаговую инструкцию.

## Мои действия
- Проанализировала исходный текст, выделила логические блоки.
- Упростила язык, убрала лишнюю информацию.
- Добавила визуальное выделение шагов и список частых ошибок.

## Результат
- Время выполнения операции сократилось с 10 до 3 минут
- Количество вопросов в поддержку снизилось на 60%

## Сравнение

❌ **Было:** сплошной текст без структуры  
✅ **Стало:** пошаговая инструкция с выделениями
