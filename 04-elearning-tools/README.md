# Разработка электронных курсов и справочных систем

## Инструменты
- Adobe Captivate (создание интерактивных курсов, симуляций, тестов)
- RoboHelp (создание справочных систем, CHM, HTML5-справки)

## Что я делаю
- Разрабатываю сценарии и структуру курсов под бизнес-задачи
- Создаю интерактивные модули: симуляции работы в SAP, тесты для проверки знаний
- Генерирую справочную систему в формате HTML5 (RoboHelp) с поиском и навигацией
- Экспортирую материалы в SCORM для загрузки в LMS

## Примеры

### 1. Интерактивный курс «Внесение данных медосмотров в SAP»
Создан в Adobe Captivate. Включает:
- Симуляцию работы в SAP (кликабельные поля)
- Всплывающие подсказки
- Контрольный тест с обратной связью

![Скриншот курса](course_screenshot.png)

### 2. Справочная система «Охрана труда в рознице»
Создана в RoboHelp. Содержит:
- Поиск по ключевым словам
- Алфавитный указатель
- Встроенные инструкции и чек-листы
