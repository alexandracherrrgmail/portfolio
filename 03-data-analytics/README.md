# Анализ рынка телекоммуникационной компании

## Задача
Провести исследование рыночных данных телеком-оператора, выявить ключевые метрики и предложить точки роста.

## Мои действия
- Собрала и очистила данные (SQL, Python pandas).
- Провела разведочный анализ (EDA) с визуализацией распределений, выбросов, корреляций.
- Сегментировала клиентскую базу, выявила профили оттока.
- Подготовила дашборд для ключевых показателей (таблица, графики).

## Результат
- Определены 3 ключевых фактора оттока, что позволило предложить точечные меры удержания.
- Подготовлена презентация для руководства с рекомендациями, оценён экономический эффект.
